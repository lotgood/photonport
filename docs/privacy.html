PhotonPort Privacy Policy

Effective date: 2026-07-10

PhotonPort has no account system, analytics service, advertising, or developer-operated relay server. Display video, system audio, input events, and pairing messages travel directly between the Mac and the user's iPhone or iPad over USB/usbmux or the local network.

Data processed

  • The Mac app processes screen pixels, system audio, cursor state, and display configuration to provide streaming.
  • The iOS app processes received media and sends touch and scroll events to the connected Mac.
  • Pairing keys are stored in Keychain. Local identifiers, names, and app preferences are stored on-device.
  • Diagnostic logs remain local, rotate at approximately 1 MB, and are never uploaded automatically.

Network services

WiFi mode uses the local network. The macOS app may contact GitHub Pages and GitHub Releases for signed updates. TestFlight and App Store distribution are operated by Apple and are subject to Apple's terms.

Collection and retention

The developer does not automatically collect or sell app data. Data stays on the participating devices unless the user deliberately shares a log or diagnostic report. Local data remains until removed, reset, or deleted.

Before posting a public issue, redact device names, identifiers, IP addresses, Bonjour names, and personal content.

Contact

Use GitHub Issues for general questions. Send sensitive reports through private vulnerability reporting.
